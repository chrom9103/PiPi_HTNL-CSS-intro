### 2.4. レイアウト
CSSの魅力はただの装飾だけではありません。レイアウトの調整も行うことができます。

#### 2.4.1. ボックスレイアウト
HTMLでは、見出し・文章・画像など、すべての要素が「箱（ボックス）」として扱われます。Webページは、この箱を積み重ねたり並べたりして作られています。
この箱の大きさや余白、配置を調整してWebページのレイアウトを作る仕組みを**ボックスレイアウト**と呼びます。

[ボックスレイアウトの基本構造]
```
┌────────────────────────────┐ ← margin（外側余白）
│                            │
│  ┌──────────────────────┐  │
│  │       border         │  │
│  │   ┌──────────────┐   │  │
│  │   │   padding    │   │  │
│  │   │              │   │  │
│  │   │  ┌────────┐  │   │  │
│  │   │  │content │  │   │  │
│  │   │  └────────┘  │   │  │
│  │   │              │   │  │
│  │   └──────────────┘   │  │
│  └──────────────────────┘  │
│                            │
└────────────────────────────┘
```

- content(内容): 文字や画像などの内容
- padding: 内容と枠線の間の余白
- border: ボックスを囲む枠線
- margin: ボックスの外側の余白

##### 2.4.1.1. 枠線 <border>
borderとは、ボックスの周囲に表示される枠線のことです。コンテンツの範囲を区切ったり、強調したりするために使用されます。
borderでは、以下のような設定が可能です。
- 線の太さ（width）
- 線の種類（style）
- 線の色（color）
これらを組み合わせることで、さまざまな見た目の枠線を作ることができます。

**線の太さ**
枠線の太さを設定します。
####　例
- 1px：細い線
- 3px：やや太い線
- 5px：強調された線
太い枠線は目立ちやすく、重要な情報を強調する場合に使用されます。
>[!TIP]
>px(ピクセル)は大きさを表す単位です。
>数値が大きいほど、余白や線も大きくなります。

```html
<div class="box1">細い線</div>
<div class="box2">太い線</div>
```

```css
.box1{
    border: 1px solid black;
}

.box2{
    border: 5px solid black; 
}
```

**線の種類**
borderでは、線の表示形式を変更できます。用途によって線の種類を変えることで、デザインに変化を付けられます。
####　例
- solid  :実線
- dashed :破線
- dotted :点線
- double :二重線

```html
<div class="solid">実線</div>
<div class="dashed">破線</div>
<div class="dotted">点線</div>
<div class="double">二重線</div>
```

```css
.solid{
    border: 3px solid black;
}

.dashed{
    border: 3px dashed black; 
}

.dotted{
    border: 3px dotted black;
}

.double{
    border: 3px double black;
}
```

**線の色**
borderの色を設定します。色を変更することで、注意喚起や強調表現ができます。
- black（黒）
- gray（灰色）
- blue（青）
- red（赤）

```html
<div class="solid">黒色の実線</div>
<div class="dashed">灰色の破線</div>
<div class="dotted">青色の点線</div>
<div class="double">赤色の二重線</div>
```

```css
.solid{
    border: 3px solid black;
}

.dashed{
    border: 3px dashed gray; 
}

.dotted{
    border: 3px dotted blue;
}

.double{
    border: 3px double red;
}
```

##### 2.4.1.2. 内側余白 <padding>
paddingとは、ボックスの内側にある余白のことです。具体的には、内容（content）と枠線（border）の間にできる空間を指します。paddingを設定することで、内容が枠線に直接くっつかず、見やすく整ったレイアウトになります。

**paddingの指定方法**

```css
.box{
    padding: 20px;
}
```

この場合、上下左右すべてに20pxの余白が入ります。

- 方向別の指定
paddingは方向ごとに細かく設定することもできます。

```css
.box{
    padding-top: 10px; /*上*/
    padding-right: 20px; /*右*/
    padding-bottom: 30px; /*下*/
    padding-left: 40px; /*左*/
}
```

また、一つにまとめて書くこともできます。

```css
.box{
    padding: 10px 20px 30px 40px;
}
```

※ 時計回りに「上 → 右 → 下 → 左」の順で指定します。

**実装例**

```html
<div class="box">paddingの例</div>
```

```css
.box{
    border: 3px solid red;
    padding: 20px;
    background-color: #0000ff;
}
```

背景色は内容部分とpadding部分に適用されます。
今回では、文字の周囲にある青色の余白部分が、paddingによって追加された領域です。

##### 2.4.1.3. 外側余白 <margin>
marginとは、ボックスの外側に作られる余白のことです。marginは、要素同士の間隔を調整するために使われます。

**marginの指定方法**
- 中央寄せ
autoを指定すると、左右の余白が自動調節されます。その結果、要素が左右中央に配置されます。
※横幅（width）が指定されていないと中央寄せになりません。

```css
.box{
    width: 200px;
    margin: 0 auto;
}
```

- 上下の余白 → `0`
- 左右の余白 → `auto`

このようにすると、左右の余白が自動で均等になり、要素が中央に配置されます。

- 一括指定

```css
.box{
    margin: 20px;
}
```

上下左右すべてに20pxの外側余白が入ります。

>[!TIP]
>marginの場所を確認してみよう！
>
>```html
><div class="container">
>    <div class="box1">BOX 1</div>
>    <div class="box2">BOX 2</div>
></div>
>```
>
>```css
>.container{
>    background-color: #808080;
>}
>
>.box1{
>    border: 3px solid red;
>    margin-bottom: 20px;
>    padding: 10px;
>    background-color: #ffffff;
>}
>
>.box2{
>    border: 3px solid blue;
>    padding: 10px;
>    background-color: #ffffff;
>}
>```

>この灰色の隙間がmarginの場所です。


- 方向別の指定

```css
.box{
    margin-top: 10px; /*上*/
    margin-right: 20px; /*右*/
    margin-bottom: 30px; /*下*/
    margin-left: 40px; /*左*/
}
```

また、paddingと同様に、一つにまとめて書くこともできます。

```css
.box{
    margin: 10px 20px 30px 40px;
}
```

※ 時計回りに「上 → 右 → 下 → 左」の順で指定します。

**実装例**

```html
<div class="box1">BOX 1</div>
<div class="box2">BOX 2</div>
```

```css
.box1{
    border: 3px solid red;
    margin-bottom: 20px;
    padding: 10px;
}
.box2{
    border: 3px solid blue;
    padding: 10px;
}
```
