### 2.6. グリッドレイアウト（`display: grid`）
グリッドレイアウトは、要素内の配置を格子状に区切って指定する方法です。
#### 2. 6. 0 事前準備
グリッドレイアウトで配置するためのコンテナと、内部に配置するアイテムを作成します。
```html
<body>
    <div class="grid">
        <div class="item" id="item1">1</div>
        <div class="item" id="item2">2</div>
        <div class="item" id="item3">3</div>
        <div class="item" id="item4">4</div>
    </div>
</body>
```
```css
.grid{
    width: 600px;
    height: 450px;
    border: 2px solid #000000;
}
.item{
    background-color: #aaaaaa;
    border: 1px solid #000000;
}
```
#### 2. 6. 1 格子状に分割する
- `display:grid`: グリッドレイアウトであることの宣言
- `grid-template-columns`: 分割する列数やサイズの指定
- `grid-template-rows`: 分割する行数やサイズの指定

セル: 分割された1マス1マスをセルといいます。

CSSに以下を追加してください。
```css
.grid{
    ...
    display: grid;
    grid-template-columns: 150px 1fr 1fr;
    grid-template-rows: 2fr 1fr;
}
```
>`fr`は指定された幅を除いた空間を、割合で分割する単位です。

この例では、横は1列目が150px、残りの空間が1:1に分割され、縦は1行目と2行目が2:1で分割されています。

>[!TIP] ブラウザの検証機能でグリッドレイアウトのhtmlタグにカーソルを合わせると、分割の境界線が表示される。

#### 2. 6 .2 配置エリアを指定する
- `grid-column`: 横方向の配置に使用する列を指定する
- `grid-row`: 縦方向の配置に使用する行を指定する
>グリッドレイアウトにおける範囲
>
>左(上)の外周を1として、分割の境界に数字を振って考えます。 つまり、n分割した際に取りうる値は、1 ~ n+1となります。 配置する範囲は左端(上端)と右端(下端)の数値を / で区切って表します。

CSSに以下を追加してください。
```css
#item1{
    grid-column: 2/4;
    grid-row: 2/3;
}
```
`item1`のidのついた要素の位置が、横方向は2本目から4本目の境界線の範囲に、縦方向は2本目から3本目の境界線の範囲に指定されます。

同様に他の要素の配置も指定することができます。
```css
#item2{
    grid-column: 2/3;
    grid-row:  1/2;
}
#item3{
    grid-column: 3/4;
    grid-row:  1/2;
}
#item4{
    grid-column: 1/2;
    grid-row:  1/3;
}
```
#### 2. 6. 3 エリア内での配置
CSSの`.item`の箇所を以下のように変更してください。
```css
.item{
    background-color: #aaaaaa;
    border: 1px solid #000000;

    width: 50px;
    height: 50px;
}
```
各アイテムがエリアの左上に偏ってしまっています。

アイテムの位置を指定するには以下の構文を使用します。
- `justify-items`: 各エリアの内部に含まれるアイテムの横方向の配置を指定する
- `align-items`: 各エリアの内部に含まれるアイテムの縦方向の配置を指定する
- `place-items`: 各エリアの内部に含まれるアイテムの縦横方向の配置を指定する
```css
.grid{
    ...
    justify-items: center;
    align-items: end;
}
```
`place-items`を用いることでまとめて書くことができます。
```css
.grid{
    ...
    place-items: end center;
}
```
>`place-items`は`align`-`justify`の順番のため注意

特定のアイテムの位置のみを指定したい場合、アイテムのstyleで個別に配置を指定します。
- `justify-self`: 自分自身の横方向の位置を指定する
- `align-self`: 自分自身の縦方向の配置を指定する
- `place-self`: 自分自身の縦横方向の配置を指定する
```css
#item3{
    ...
    justify-self: end;
    align-self: start;
}
```
```css
#item3{
    ...
    place-self: start end;
}
